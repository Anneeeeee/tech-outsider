HTML 101

HTML stands for HyperText Markup Language. It's written for web browsers with interaction. HTML describes the structure of a web page.

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Browsers do not display the HTML tags, but use them to interpret the content of the page. Contents include the following:

Text

Media

HTML tags

An HTML element is set off from other text in a document by "tags".

Tags consist of the element name surrounded by "<" and ">". Many browsers have default styles for the appearance of the document based on the meaning of the tag.
HTML5, the current standard for HTML, has roughly 100 tags.
